# Automatic-animal-feeder
פרוייקט סיום של התיכון. הפרוייקט התבצע בזוגות. מצורפת חוברת של הפרוייקט(חוברת יב.PDF) שבה רשומים כל ההסברים על הפרוייקט.
הפרוייקט התבצע בעזרת ARDUINO.

שם הפרוייקט: מאכיל חיות אוטומטי

 הצגת הבעיה: לאנשים יש בעיה בכך שיש להם בעלי חיים ועבודה עד שעות מאוחרות והם לא יכולים לחזור הביתה רק בשביל להאכיל את החיה שלהם, לפעמים האנשים נוסעים לנסיעות ולא תמיד יש להם אמון שהבן אדם עליו השאירו את החיה לא יישכח להאכיל אותה בזמן... 

תיאור פרוייקט: בעזרת לוח המקשים אנו קובעים את הזמן להאכלת החיה והזמן מוצג לנו על המסך. כשמגיע הזמן שהתקבל הפלטפורמה זזה והאוכל נופל בפתח שיש בפלטפורמה כמה שניות ואז היא עוד פעם זזה והפתח נסגר. כך החיה מקבלת אוטומטית בזמן הקבוע את האוכל שלה.
